# 桌面默认本地端口固定为 47240

ADR-0010 要求的「公开默认端口」定为 **47240**：位于用户端口段（1024–49151）、低于 Windows/macOS 动态端口起点 49152，未被 IANA 注册占用，与常见开发端口（3000/8000/8080/5173 等）无冲突。自托管/compose 形态维持 8000 不变。该常量同时存在于 sidecar CLI 默认值（`sag_api/sidecar.py::DEFAULT_DESKTOP_PORT`）与壳的端口选择逻辑，二者必须同步修改。用户在设置中修改端口后持久化于壳侧偏好，冲突时按 ADR-0010 显式修复，绝不静默漂移。
