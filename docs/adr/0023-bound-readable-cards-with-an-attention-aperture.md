# 使用阅读孔径约束可读卡片

> 已由 [ADR 0027](0027-use-focus-relation-slices-and-mode-specific-layouts.md) 取代。

知识宇宙将事件窗口与卡片预览分开：窗口内的事项、实体和关系始终作为完整网络参与渲染，只有处于安全视区、合适深度或当前关系焦点中的事项竞争有限的卡片预览名额，相关实体卡片随事项和聚焦关系展示，不另设预览数量。悬停时高亮完整的一跳关系，但由聚焦网络接管而非突破预览上限；当前节点和优先邻居显示卡片，其余邻居保持高亮图元并汇总溢出数量。关闭卡片时停止自动预览，悬停或锁定仍可临时呈现当前关系。这样提高窗口只会增加星点和关系密度，不会造成卡片无限叠加。
