# 两种知识宇宙状态共用一套配置

时空探索态与线索累积态共用一套事件级知识宇宙配置，不为两种状态复制参数。默认缓存 1000 个事项、窗口显示 50 个事项、启用事项与实体卡片并预览 10 个事项；时空探索默认每页加载 20 个事项，在当前窗口前后各预取至多 3 页，按距离由近到远串行请求，同一时刻只加载一页。分页只向缓存补充数据，窗口采用连续增量过渡，稳定后仍为 50 个事项，过渡期至多额外保留一页作为进出缓冲。相关实体始终随事项完整进入；线索累积态不执行时序分页或预取，新知识只来自搜索、问答和主动拓展。状态仅决定同一批事实采用时间纵深布局还是稳定关系布局。系统保留设备安全上限，并在配置值无法完全生效时明确展示实际值与原因。
